RYDEX – Blaze Ride Booking Platform

Description:  
RYDEX is a real‑time ride booking platform designed with secure authentication, live trip tracking, and seamless payment integration. It enables riders to book, track, and complete trips with OTP‑based verification while partners onboard through video KYC and document uploads.


Tech Stack:  
Next.js | TypeScript | MongoDB | Redux Toolkit | Socket.IO | Cloudinary | Razorpay | Leaflet | ZegoCloud | Tailwind CSS 

Key Features:

🔐 Secure Login & Verification – Google/OTP login, start & end OTP for fraud prevention. <br>
🗺️ Live Ride Tracking – Real‑time map updates using Leaflet & React‑Leaflet. <br>
👥 Partner Onboarding – Car & bank detail uploads, video KYC via ZegoCloud.<br>
💳 Payments – Fast, secure transactions powered by Razorpay.<br>
📸 Media Handling – Cloudinary integration for image & media uploads.<br>
⚡ Performance – Optimized with Next.js, TypeScript, and Redux Toolkit for scalability.<br>
